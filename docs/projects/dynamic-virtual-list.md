# 🔥 dynamic-virtual-list

> 高性能 Vue 3 虚拟滚动组件，TypeScript 编写，零第三方依赖

## 📝 简介

一个专注于**极致性能**和**易用性**的虚拟滚动组件，灵感来自业界主流方案，针对 Vue 3 Composition API 做了深度优化。

## 🛠 技术栈

- **Vue 3** (Composition API)
- **TypeScript**（严格模式）
- **ResizeObserver**（动态尺寸监听）
- **零第三方依赖**（除 Vue 自身）

## 🚀 在线 Demo

🌐 <https://hhcompile.github.io/dynamic-virtual-list/>

## 📦 源码仓库

<https://github.com/HhCompile/dynamic-virtual-list>

## ✨ 核心特性

- ✅ **万级数据流畅滚动**：实测 10w+ 数据无卡顿
- ✅ **固定高度 / 动态高度**双模式
- ✅ **TypeScript 严格类型**保障
- ✅ **零外部依赖**，包体积 < 10KB (gzipped)
- ✅ **支持横向 / 纵向**滚动
- ✅ **SSR 友好**
