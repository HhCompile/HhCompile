# 📋 json-schema-form-engine

> JSON Schema 驱动的可视化表单引擎，渲染/校验/拖拽设计一体化

## 📝 简介

通过 JSON Schema 配置即可生成完整表单，集成校验、拖拽设计器、字段联动等高级能力。

## 🛠 技术栈

- **Vue 3** (Composition API)
- **TypeScript**
- **JSON Schema** (Draft 7 / 2019-09)
- **拖拽引擎** (vue-draggable-next)

## ✨ 核心特性

- ✅ **JSON Schema 驱动**：一份 schema 生成表单
- ✅ **22 种表单组件**：input / select / date / upload / cascader / tree ...
- ✅ **可视化拖拽设计器**：所见即所得
- ✅ **完整校验体系**：必填 / 长度 / 正则 / 自定义
- ✅ **字段联动**：show / hide / required 响应式
- ✅ **TypeScript 类型安全**
