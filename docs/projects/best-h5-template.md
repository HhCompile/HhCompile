# 📱 best-h5-template

> Vue 3 + TypeScript H5 移动端模板，开箱即用

## 📝 简介

针对企业级 H5 项目设计的脚手架模板，集成移动端适配、路由、状态管理、API 封装等常用能力。

## 🛠 技术栈

- **Vue 3** (Composition API)
- **TypeScript**
- **Vite** 5
- **Vant UI** 4 (移动端组件库)
- **Pinia** (状态管理)
- **Vue Router** 4
