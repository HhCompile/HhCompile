# ⚛️ react-component-kit

> React × TypeScript 组件库（含 Storybook + Vitest）

## 📝 简介

一个从零搭建的 React 组件库项目骨架，集成 Storybook 文档站、Vitest 单元测试、Rollup 库模式打包。

## 🛠 技术栈

- **React** 18
- **TypeScript**
- **Vite** 5（开发环境）
- **Storybook** 7（组件文档）
- **Vitest**（单元测试）
- **Rollup**（库模式打包）
