# 🏠 HhCompile.github.io

> 个人作品集站点，Vue 3 + Vite，GitHub Actions 自动部署

## 📝 简介

简洁的 SPA 单页应用，展示个人简介、技能、项目、联系方式等内容。

## 🛠 技术栈

- **Vue 3** (Composition API)
- **TypeScript**
- **Vite** 8
- **纯 CSS 暗色主题**（无 UI 库依赖）
- **GitHub Actions** 自动部署

## 🚀 在线 Demo

🌐 <https://hhcompile.github.io>

## 📦 源码仓库

<https://github.com/HhCompile/HhCompile.github.io>

## ✨ 核心特性

- ✅ **6 大区块**：Hero / About / Skills / Experience / Projects / Contact
- ✅ **纯 CSS 暗色主题**：无任何 UI 库
- ✅ **GitHub Pages 自动化部署**
- ✅ **vue-tsc 严格类型检查**
